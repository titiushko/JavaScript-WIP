jQuery Validator

Este formulario utiliza un sencillo plugin de validación.

No se puede enviar el formulario hasta que los campos requeridos estén completados.

Si los campos requeridos no están completados y se intenta enviar el formulario se mostrarán los campos que necesitan ser rellenados y el primero de ellos obtendrá el foco.
